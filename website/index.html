About HueGrab

HueGrab was built with a simple philosophy: color picking should be effortless. Whether you're a designer matching colors from physical objects, an artist seeking inspiration, or a developer needing exact color codes, HueGrab makes it instant.

Built with React Native and Expo, HueGrab is open source and privacy-focused. No ads, no tracking, no data collection. Just a simple tool that does one thing well.
